# HC Dashboard 🚀

> **⚠️ Work in Progress**: This project is currently under active development. Some features may be incomplete or subject to change.

A simple and lightweight Android dashboard designed to visualize health and fitness data stored in **Android Health Connect**.

## Overview

The goal is to provide a simple visualization of your health data stored in Android Health Connect.

## Tech Stack

- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI**: [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Health Data**: [Health Connect API](https://developer.android.com/health-and-fitness/guides/health-connect)
- **Charts**: [Vico](https://github.com/patrykandpatrick/vico)

## Requirements

- Android device with **Health Connect** installed and configured.
- Permissions granted within the app to read specific health data types.
